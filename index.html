Meshki

Stylish. Black. Responsive.


Design your whole UI in absolutely no time. We have done our best to keep Meshki stupidly simple and clean.

Code while smiling. Meshki is easy to use and edit. Just grab a text editor and start building your web pages in total elegance.

Oh did I say it's super fast? By using less code and less complex logics, Meshki is light-weight and is not resource-hungry.

What is Meshki?

Meshki is a simple, black-colored, responsive boilerplate to kickstart any responsive project. It is only ~10 KiB (minified) and ~23 KiB (normal), including both CSS and JS files, which makes it superb for small to medium projects. It's easy to use, clean and is just beautiful. It started as a fork of Skeleton project, and it still uses some of the codes of it, Meshki is considered to be a total overhaul of Skeleton and it barely resembles it when you take a look at it. Don't be afraid to have a look at Meshki's code! If you spot any bugs, or you want some improvements, head to our GitHub repository and open an issue or a pull request!

Components

Meshki has various ready-to-use components which you can create anywhere you like. Here are a list of components and how you can put them into use:

Buttons

You can create buttons easily with <a>, <button>, <input type="submit">, <input type="reset"> and <input type="button">. There are also five different types of buttons:

Simple

Here's the code for them:

            
    <a class="button">Simple</a>
    <button class="default">Default</button>
    <input type="submit" class="success" value="Success" />
    <input type="reset" class="danger" value="Danger" />
    <input type="button" class="warning" value="Warning" />

The Grid System

The grid is a 12-column fluid grid with a max width of 960px, that shrinks with the browser/device at smaller sizes. The max width can be changed with one line of CSS and all columns will resize accordingly. The syntax is simple and it makes coding responsive much easier. Go ahead, resize the browser.

Here's the code for how to use the grid system:

      <div class="one column">
        A div with one column length
      </div>
      <div class="eleven columns">
        A div with eleven columns length
      </div>
    </div>

    <div class="row">

        A div with one-third of the screen's length
      </div>

      </div>
    </div>

    <div class="row">
      <div class="one-half column">

      </div>

Form Elements

Meshki also has customized form elemets, which are just as beautiful as the others. Here are some examples of form elements you can use:

This is the code snippet on how you can use form elements: 

            
    <form>
      <!--Text Inputs -->
      <div class="row">
        <div class="six columns">
          <label for="user1">Username:</label>
          <input id="user1" class="expand" type="text" name="user1" placeholder="Enter text here">
        </div>
        <div class="six columns">
          <label for="pass1">Password:</label>
          <input id="pass1" class="expand" type="password" name="pass1">
        </div>
      </div>

      <!--Disabled Text Inputs -->
      <div class="row">
        <div class="twelve columns">
          <label for="dis-email1">Email:</label>
          <input id=dis-email1"" class="expand" type="email" name="dis-email1" value="me@borderliner.ir" disabled>
        </div>
      </div>

      <!--Text Area -->
      <div class="row">
        <div class="twelve columns">
          <textarea class="expand">This is a sample <textarea> element.</textarea>
        </div>
      </div>

      <!--Dropdown -->
      <div class="row">
        <div class="six columns">
          <select class="expand">
            <option selected disabled hidden>Choose your country</option>
            <option value="us">United Stated</option>
            <option value="ca">Canada</option>
            <option value="de">Germany</option>
            <option value="jp">Japan</option>
          </select>
        </div>
      </div>

      <!--Checkboxes and Radio Buttons -->
      <div class="row">
        <div class="six columns">
          <input id="rad-male" type="radio" name="gender" value="male">
          <label for="rad-male" class="radio">Male</label>
          <input id="rad-female" type="radio" name="gender" value="female">
          <label for="rad-female" class="radio">Female</label>
          <input id="rad-other" type="radio" name="gender" value="other">
          <label for="rad-other" class="radio">Other</label>
        </div>
        <div class="six columns">
          <input id="chk-eula" type="checkbox" name="chk-eula">
          <label for="chk-eula" class="checkbox">I've read and agree with Terms of Service.</label>
          <input id="chk-news" type="checkbox" name="chk-news">
          <label for="chk-news" class="checkbox">I wish to receive newsletters from this site.</label>
        </div>
      </div>
      
      <div class="row">
        <button>Submit</button>
      </div>
    </form>
