What is Meshki?

Meshki is a simple, black-colored, responsive boilerplate to kickstart any responsive project. It is only ~10 KiB (minified) and ~23 KiB (normal), including both CSS and JS files, which makes it superb for small to medium projects. It's easy to use, clean and is just beautiful.
